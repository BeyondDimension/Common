Enum ArchitectureFlags

Namespace
System.Runtime.InteropServices

BD.Common8.Bcl.dll

CPU 架构(处理器体系结构)，例如 x86、Arm 等 Architecture 例如不同的 Android 设备使用不同的 CPU，而不同的 CPU 支持不同的指令集。

CPU 与指令集的每种组合都有专属的应用二进制接口 (ABI)。

https://developer.android.google.cn/ndk/guides/abis.html

在比较版本支持的值与设备支持的值中的交集中，值越大的，优先选取
