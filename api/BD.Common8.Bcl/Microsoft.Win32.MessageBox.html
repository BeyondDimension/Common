Class MessageBox

Namespace
Microsoft.Win32
Assembly
BD.Common8.Bcl.dll

显示消息窗口（也称为对话框），向用户显示消息。 这是一个模式窗口，可阻止应用程序中的其他操作，直到用户将其关闭。 MessageBox 可包含通知并指示用户的文本、按钮和符号。

  • Windows 上使用 MessageBoxW 函数 (winuser.h) https://learn.microsoft.com/zh-cn/windows/win32/api/winuser/nf-winuser-messageboxw
  • macOS 上使用 NSAlert https://developer.apple.com/documentation/appkit/nsalert
