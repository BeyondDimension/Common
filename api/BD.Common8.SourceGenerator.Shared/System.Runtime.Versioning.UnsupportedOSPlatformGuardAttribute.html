Remarks

Callers can apply a UnsupportedOSPlatformGuardAttribute to a field, property or method and use that field, property or method in a conditional or assert statements as a guard to safely call APIs unsupported on those platforms.

The type of the field or property should be boolean, the method return type should be boolean in order to be used as platform guard.
