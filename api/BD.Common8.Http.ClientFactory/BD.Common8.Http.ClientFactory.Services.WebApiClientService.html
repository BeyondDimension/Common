CreateClient()

Creates and configures an HttpClient instance using the configuration that corresponds to the logical name specified by name.

protected abstract HttpClient CreateClient()

Returns

HttpClient

A new HttpClient instance.

Remarks

Each call to CreateClient(string, HttpHandlerCategory) is guaranteed to return a new HttpClient instance. It is generally not necessary to dispose of the HttpClient as the IClientHttpClientFactory tracks and disposes resources used by the HttpClient.

Callers are also free to mutate the returned HttpClient instance's public properties as desired.
